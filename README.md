# 🐱 Catmagotchi

A cozy cat platformer + virtual pet hybrid! Take care of your adorable pixel cat **Mitten** at home, then head out on an endless adventure through lush forests, crystal springs, and scenic meadows.

## 🎮 Play Now

**[Play Catmagotchi](https://HelenCAblaza.github.io/catmagotchi/)**

## 🏠 Home Mode

Keep Mitten happy and healthy by managing their stats:
- 🍖 **Hunger** — Feed fish to keep the tummy full
- 😸 **Happiness** — Play with yarn balls for joy
- ⚡ **Energy** — Let Mitten sleep to recharge
- 🧼 **Cleanliness** — Give baths to stay fresh

## 🌍 Adventure Mode

Explore an **endless scrolling world** with beautiful pixel art scenery:
- 🌲 **Pine, oak, and poplar trees** — lush forest canopy
- 🪷 **Ponds** — lily ponds, natural pools, duck ponds, and crystal springs
- 🌸 **Flowers, bushes, and rocks** — scattered natural details
- 🐟 **Collect fish** — floating treats to feed Mitten later
- 🧶 **Collect yarn balls** — toys for playtime back home
- 📱 **Virtual joystick** — smooth round touch controls for mobile

### Controls
| Input | Action |
|-------|--------|
| ← / → | Move left/right |
| ↑ | Jump |
| Mouse / Touch | Click buttons to feed, play, sleep |
| Round Joystick | Mobile movement (appears on touch) |

## 🛠️ Tech Stack

- [Phaser 3](https://phaser.io/) — 2D game engine
- HTML5 Canvas
- Pixel art assets (XPM + ImageMagick generation)
- GitHub Pages hosting

## 📦 Development

```bash
git clone https://github.com/HelenCAblaza/catmagotchi.git
cd catmagotchi
# Open index.html in any browser
```

Or serve locally:
```bash

# or
python3 -m http.server 8080
```
