# 🐱 Catmagotchi

A cat platformer + tamagotchi hybrid game! Jump through levels collecting fish and toys, then come home to feed, play with, and care for your virtual cat.

## 🎮 Play Now

**[Play Catmagotchi](https://HelenCAblaza.github.io/catmagotchi/)**

## 🎯 Gameplay

- **Home** 🏠: Take care of Mittens! Feed fish, play with yarn, and let them sleep.
- **Adventure** 🌍: Platform through levels collecting fish and toys.
- **Stats**: Keep Hunger, Happiness, Energy, and Hygiene high or Mittens gets grumpy!

### Controls
- **← / →** — Move left/right
- **↑** — Jump
- **Mouse** — Click buttons to feed, play, sleep

## 🛠️ Tech Stack

- [Phaser 3](https://phaser.io/) — 2D game engine
- HTML5 Canvas
- GitHub Pages hosting

## 📦 Development

```bash
git clone https://github.com/HelenCAblaza/catmagotchi.git
cd catmagotchi
# Open index.html in any browser
```

Or serve locally:
```bash
npx serve .
# or
python3 -m http.server 8080
```
